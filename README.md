# git-course-pr-example

Жили у бабуси  
Два весёлых гуся:  
Один — серый,  
Другой — белый,  
Два весёлых гуся.  

Мыли гуси лапки в луже у канавки,
Один серый, другой белый
Спрятались в канавке.
Один серый, другой белый
Спрятались в канавке.

Вот кричит бабуся -
Ой, пропали гуси,
Один серый, другой белый,
Гуси, мои гуси.
Один серый, другой белый,
Гуси, мои гуси.

Выходили гуси, кланялись бабусе,
Один серый, другой белый
Кланялись бабусе.
Один серый, другой белый
Кланялись бабусе.
